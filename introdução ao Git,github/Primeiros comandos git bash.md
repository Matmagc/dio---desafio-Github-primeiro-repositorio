# comandos iniciais do git

## -git init

### usado para iniciar o git no repositorio em questão.

## -git add

### usado para adicionar pastas e arquivos no git

## -git commit

### inicia um commmit onde ficara gravado o que foi mudado nessa atualização

## -a

### mostra arquivos ocultos

## -git config --global user.email "seu email"

### configura o email que sera usado como principal no github.

## -git config --global user.name seu_apelido

### configura o apelido que será usado no github.
