# Windows

## -cd

### navega para um repositorio especifico, -cd.. volta um repositorio.

## -cls

### limpa a tela.

## -dir

### lista todos os repositórios

## TAB

### apertar a tecla TAB autocompleta os comandos.
 
##-mkdir

### cria um repositorio.

## -del/-rmdir

### -del deleta todos os arquivos do repositório, -rmdir deleta o repositório.

## seta para cima

### ao apertar a seta para cima do teclado ele volta os comandos ja utilizados.

# Unix

## -cd

### navega para um repositorio especifico, -cd.. volta um repositorio.

## -clear

### limpa a tela, pode-se usar ctrl+l também.

## -ls

### lista todos os repositórios

## TAB

### apertar a tecla TAB autocompleta os comandos.

## -mkdir

### cria um repositorio.

## -rm -rf

### -rm vai deletar todos os arquivos da pasta, -rf digitado junto faz com que delete todos os repositorios dentro do repositório.
